An Augmented Dataset for Vision-based Unmanned Aerial Vehicles Detection and Tracking

Venue: IEEE AIPR 2023 DOI: 10.1109/AIPR60534.2023.10440657 Project Type: Dataset + Benchmark
Back to Projects Code/Dataset Demo Paper Presentation

Problem Statement

Vision-based UAV detection and tracking needs high-quality data that represents realistic weak-target, moving-camera, and cluttered scenarios.

Research Gap

Public resources for UAV-to-UAV detection/tracking are limited in diversity and difficulty, making it hard to evaluate robustness consistently across challenging operating conditions.

Methodology

The work builds an augmented dataset and benchmark workflow to support robust research on weak-target perception and temporal visual analysis in airborne settings.

  • Curated and augmented UAV image sequences for detection/tracking research
  • Designed to support reproducible model evaluation under practical challenges
  • Provides a shared base for comparing methods under similar conditions
  • Public release through project repository for wider community use

Results and Impact

The dataset supports repeatable benchmarking for weak-target UAV detection/tracking and was published in IEEE AIPR 2023.

Qualitative Comparison

Compared with smaller or less varied UAV resources, this augmented dataset provides broader scene variation and stronger stress conditions for evaluating detection/tracking robustness.
