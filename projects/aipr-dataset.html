An Augmented Dataset for Vision-Based Unmanned Aerial Vehicles Detection and Tracking

Venue: IEEE AIPR 2023 Type: Dataset + Benchmark DOI: 10.1109/AIPR60534.2023.10440657 Dataset Engineering Reproducible Evaluation
Back to Projects Paper Code / Dataset Demo

Problem

UAV detection/tracking research needs benchmarks that include realistic weak-target behavior, clutter, and motion variability.

Why It Matters

Robust evaluation quality depends on challenging and reproducible data conditions, not only model architecture improvements.

Method

The project curates and augments UAV sequences to support detection and tracking experiments under practical visual stress.

Key Contributions

  • Augmented UAV sequence curation for detection and tracking tasks
  • Benchmark design for stress-oriented model comparison
  • Public release pathway for reusable evaluation and replication

Results / Impact

Published at IEEE AIPR 2023, this dataset supports reproducible UAV detection/tracking benchmarking under practical scene complexity.

Relative to smaller resources, the augmented benchmark provides broader stress conditions for robustness-oriented evaluation.

My Role

I led dataset design, augmentation strategy, benchmark framing, and publication delivery.
