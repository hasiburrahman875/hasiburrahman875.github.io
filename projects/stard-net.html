STARD-Net: SpatioTemporal Attention for Robust Detection of Tiny Airborne Objects from Moving Drones

Venue: ACM TSAS Publication Year: 2026 DOI: 10.1145/3787467
Back to Projects Code Demo Paper Presentation

Problem Statement

Detecting tiny airborne objects from moving drone platforms is difficult due to object scale, camera motion, cluttered backgrounds, camouflage effects, and intermittent occlusions.

Research Gap

Many frame-centric detectors miss weak targets and underuse temporal continuity. This leads to unstable predictions when appearance cues are faint or quickly changing.

Methodology

STARD-Net integrates spatial feature enhancement, temporal sequence modeling, and spatiotemporal attention to improve tiny-target detection reliability in dynamic drone scenes.

  • Drone Receptive Field Block (DRFB) for multiscale spatial feature extraction
  • ConvLSTM temporal module for sequence-level motion context
  • SpatioTemporal Attention Block (STAB) for contextual aggregation
  • Transformer-enhanced detection head for robust final prediction

Results

The paper reports significant performance gains across multiple public airborne object datasets while maintaining real-time inference behavior.

Qualitative Comparison

Compared with single-frame or weak temporal baselines, STARD-Net better preserves tiny object continuity in cluttered and motion-heavy clips, reducing missed detections for faint airborne targets.
