STARD-Net

STARD-Net: Spatio-Temporal Attention Residual Dilated Network for UAV-Based Airborne Object Detection

STARD-Net addresses tiny airborne object detection from moving UAV cameras, where targets are small, visually weak, and frequently affected by clutter, camera motion, camouflage, and partial occlusion. The project is framed as computer vision and spatiotemporal machine learning for robust aerial perception.

Area: Computer Vision · Small-Object Detection · Spatiotemporal Modeling Venue: ACM Transactions on Spatial Algorithms and Systems Year: 2026

Problem Statement

Detecting airborne objects from drone-mounted cameras is difficult because the target may occupy only a few pixels, background clutter can dominate the frame, and camera/target motion creates unstable visual evidence.

Motivation

Frame-level detection can fail when the object is faint or partially occluded. STARD-Net uses temporal context and attention-driven feature modeling to improve weak target recovery in UAV-based airborne object detection.

Methodology Overview

At a high level, STARD-Net combines residual/dilated feature extraction, attention-based feature refinement, and spatiotemporal modeling. The method uses temporal context to recover weak target evidence that may not be reliable in a single frame.

  • Spatiotemporal modeling for sequence-level evidence aggregation.
  • Attention-based feature refinement for cluttered visual evidence.
  • Residual and dilated feature extraction for small target representation.
  • Temporal context for weak target recovery under motion and occlusion.

Key Contributions

  • Spatiotemporal attention for tiny airborne object detection from moving drones.
  • Feature refinement for weak and cluttered visual evidence.
  • Temporal context modeling for improved robustness under motion and occlusion.
  • UAV-oriented perception setting involving small targets and challenging backgrounds.
